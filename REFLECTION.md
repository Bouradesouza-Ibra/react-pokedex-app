# Reflection

## Where did you struggle the most?

I struggled the most with fetching data from the PokeAPI and learning how useEffect works. At first, I did not understand how React loads data after the page opens.

## Step-by-step, how did you debug / research the solution?

First, I checked the browser console to look for errors. Then I tested the PokeAPI link in the browser to make sure the data worked correctly. After that, I used console.log to understand the data better. I also reviewed examples of useState, useEffect, and fetch from React and MDN documentation.

## Did you peek at the last-resort sample repo?

No, I did not copy code from the sample repo. I used the project instructions, documentation, and my own understanding to build the app.

## One improvement with more time

With more time, I would add a favorites feature using localStorage and create a better Pokémon detail page with more stats and animations.
